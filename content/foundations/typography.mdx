| Role | Font | Weights | Why |
| --- | --- | --- | --- |
| Display & headlines | **Plus Jakarta Sans** | 600 | Geometric enough to feel modern, soft enough to feel warm. The name nods to Southeast Asia. |
| Body & UI | **Inter** | 400 / 500 / 600 | Designed for screens: tall x-height, open apertures. For teachers scanning between classes, readability is kindness. |

## Type scale

| Step | Size | Font |
| --- | --- | --- |
| Display | 120 / 96 / 72 / 48px | Plus Jakarta Sans 600 |
| Heading 1 | 32px | Plus Jakarta Sans 600 |
| Heading 2 | 24px | Plus Jakarta Sans 600 |
| Heading 3 | 20px | Plus Jakarta Sans 600 |
| Body Large | 18px | Inter 400 |
| Body | 16px | Inter 400 |
| Body Small | 14px | Inter 400 |
| Caption | 12px | Inter 500 |
| Label | 11px | Inter 600, sentence case |

Body line-height: 1.5–1.6. Related controls: TYP-1, TYP-2, TYP-3, TYP-4.

## Usage

**Do:** Plus Jakarta Sans exclusively for display and headlines · Inter for all body and UI text · generous line height · respect the scale.

**Don't:** mix display and body roles · decorative or script fonts · body text below 14px · all-caps text — use sentence case (genuine acronyms excepted).
